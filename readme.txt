Search as list
--------------

This is a version of

https://addons.thunderbird.net/thunderbird/addon/search-as-list/

updated with the necessary changes in

https://wiki.mozilla.org/Thunderbird/Add-ons_Guide_57

to work with Thunderbird 60.

To install or update previous version:

1. Open Firefox to

https://github.com/morat523035/SearchAsList

2. Download ZIP (green "Clone or download" button)
3. Extract the downloaded zip file
4. Enter the SearchAsList-master directory
5. Select all in that directory and create a zip file
6. Open Thunderbird
7. Drag the zip file into the Addons Manager tab
8. Check version is 3.1.1
